# 🏥 Hospital Management System (MERN Stack)

A full-stack **Hospital Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. This application helps hospitals efficiently manage patients, doctors, appointments, laboratory records, medicines, prescriptions, and administrative tasks through a modern web interface.

---

## 🚀 Features

### 👤 User Authentication

* Secure Login
* JWT Authentication
* Protected Routes

### 🩺 Patient Management

* Add New Patient
* Update Patient Details
* Delete Patient
* Search Patients

### 👨‍⚕️ Doctor Management

* Add Doctor
* View Doctor Details
* Update Doctor Information

### 📅 Appointment Management

* Book Appointment
* Update Appointment
* Cancel Appointment

### 💊 Medicine Management

* Add Medicines
* Medicine Inventory
* Medicine Search

### 🧪 Laboratory Management

* Create Lab Records
* View Test Results
* Update Laboratory Information

### 📄 Prescription Management

* Create Prescription
* View Prescriptions

### 📊 Dashboard

* Hospital Statistics
* Patient Overview
* Appointment Overview

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/soujanya-a-123/Hospital_Management_using_MERN.git
```

### Move into Project

```bash
cd Hospital_Management_using_MERN
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

Open another terminal

```bash
cd frontend
npm install
npm start
```
